Text is responsive

Fonts are responsive. They scale fluently between mobile and pc sizes in font size and line height.

Readability

The optimal line lenght for readability is around 60 characters. If the line is too long the users eye will have a hard time focussing on the text. This is because the length makes it difficult to get an idea of where the line starts and ends.
If a line is too short, the eye will have to travel back too often, breaking the reader’s rhythm. Too short lines also tend to stress people, making them begin on the next line before finishing the current one (hence skipping potentially important words).

Source: http://baymard.com/blog/line-length-readability

Use the class .readability to make sure the text is always at an optimal length. If you use Sass you can use @include readability() to include the needed styles.

Sizes

There are default different font sizes based on the Material Design documents. All classes start with .text- followed by the name of the font-size. If you use Sass you can include the font properties by using @include text($name).
