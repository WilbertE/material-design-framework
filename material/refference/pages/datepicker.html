Datepicker

Textfield can show a datepicker when a date is required. The datepicker is automatically generated once attached with Javascript. 

<fieldset>
    <input type="text" value="" name="date" id="date" />
    <label for="date">Date</label>
</fieldset>
$(function () {
    $("input[name='date']").mDatepicker();
});

Initializing

To initialize the components use the following function.

$(selector).mDatepicker();
Options

The following options are available:

activeDate: new Date()
The default date that is active in the datepicker
showMonth: new Date()
The month (and year) that is shown at start
rippleColor: "#cf5260"
The color of the ripple effects whe the user clicks an element inside the datepicker
daynamesShort: ["zo", "ma", "di", "wo", "do", "vr", "za"]
The short names of the days, shown in the header of the datepicker
daynamesLetter: ["z", "m", "d", "w", "d", "v", "z"]
The dayletters shown in the picker above the dates
monthnamesShort: ["jan", "feb", "mrt", "apr", "mei", "jun", "jul", "aug", "sep", "okt", "nov", "dec"]
The short monthnames shown in the header
monthnamesLong: ["januari", "februari", "maart", "april", "mei", "juni", "juli", "augustus", "september", "oktober", "november", "december"]
The monthnames shown in the button to select the months
minDate: null
The minimal selectable date in the picker
maxDate: null
The maximum selectable date in the picker
selectOnClick: false
If true the value will be updated when the user clicks a date in the datepicker and the datepicker will close. If false the user has to click 'OK' or 'Cancel' after picking a date.
closeOnClickoutside: true
Datepicker will close when the user clicks outside the datepicker
returnFormat: 'D-M-YYYY'
The format that the datepicker should return in the value of the text input
Methods

Interacting with the component from outside:

$(selector).data("mDatepicker").show();
Shows the datepicker
$(selector).data("mDatepicker").close();
Hides the datepicker (if active)
