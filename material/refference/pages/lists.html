Expandable lists

List can have expandable bodies which expands when the user clicks the list item. Use the class list--only-one-expandable to have at max only one item expanded. Expandable lists use javascript so they need to be initialized by using $(selector).mList(). Use $(selector).data("mList").toggleExpand([list item]) to programmatically toggle an item.
