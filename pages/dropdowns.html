Dropdowns

Dropdown buttons are converted from standard <select></select> and their value is also updated on change. 

<fieldset>
    <select name="select1">
        <option value="1">Option 1</option>
        <option value="2">Option 2</option>
        <option value="3">Option 3</option>
        <option selected="selected" value="4">Option 4</option>
    </select>
</fieldset>

Helpers

Helpers are small pieces of text that appear below a dropdown and help users explain what should be filled in.

Helper text
<fieldset>
    <select name="select1">
        <option value="1">Option 1</option>
        <option value="2">Option 2</option>
        <option value="3">Option 3</option>
        <option selected="selected" value="4">Option 4</option>
    </select>
    <div class="input-helper">Helper text</div>
</fieldset>

Filled and Bordered

When wrapping the field and label inside a div with the class input-filled the input uses a different background. Wrapping it inside a div with the class input-bordered it changes to a bordered mode.

Helper text
Helper text
<fieldset class="input-filled">
    <div class="input-filled__inner">
        <select name="select1">
            <option value="1">Option 1</option>
            <option value="2">Option 2</option>
            <option value="3">Option 3</option>
            <option selected="selected" value="4">Option 4</option>
        </select>
        <label for="input-label">Label</label>
        <div class="input-helper">Helper text</div>
    </div>
</fieldset>

Initializing

To initialize the components use the following function.

$("select").mDropdown();
Methods

Interacting with the component from outside:

$(selector).data("mDropdown").val(value);
Updates the component (and the original hidden <select>) to the given value
$(selector).data("mDropdown").disable();
Disables the component
$(selector).data("mDropdown").enable();
Enables the component
