Bottom sheet

Bottom sheets slide up from the bottom of the screen to reveal more content. Bottom sheets are displayed only as a result of a user-initiated action. An example sits on the bottom of this page. 

<section class="m-bottom-sheet m-bottom-sheet--large">
    <div class="m-bottom-sheet__title">
        Title
    </div>
    <div class="m-bottom-sheet__list">
        <a href="#" class="m-bottom-sheet__list-item">
            <i class="m-bottom-sheet__list-icon material-icons">image</i>
            <span class="m-bottom-sheet__list-label">Images</span>
        </a>
    </div>
</section>

Sizing

 The width of the bottom sheet can be adjusted using the classes .m-bottom-sheet--small, .m-bottom-sheet--medium and .m-bottom-sheet--large

Initializing

To initialize the component use the following function with optional options.

$(selector).mBottomSheet(options);
Options

The following options are available:

titleToggle: true
If the component has a title this will enable or disable maximizing or minimizing on title click
overlay: false
Add an dark overlay which cannot be clicked through
overlayOnlyOnMaximize: false
Add an dark overlay which cannot be clicked through and only shows when the bottom sheet get maximized
followNavigation: false
Follow the anchor clicks to its location
closeOnAnchorClick: false
Close the element when an anchor inside it is clicked
minimizeOnAnchorClick: true
Minimizes the element if an anchor inside it is clicked. If no title exists it will close.
onMinimize: function(){}
Function which get called on minimizing
onClose: function(){}
Function which get called on closing
onRemove: function(){}
Function which get called on removing the element from the DOM
onInit: function(){}
Function which get called on initialize
Methods

Interacting with the component from outside:

$(selector).data("mBottomSheet").show()
Shows the full body of the bottom sheet.
$(selector).data("mBottomSheet").hide()
Hides the full body of the bottom sheet with the header still visible if no header is visible the bottom sheet goes off screen
$(selector).data("mBottomSheet").close()
Bottom sheets goes off screen but still exists. Can be showed again by calling hide() or show()
$(selector).data("mBottomSheet").remove()
Bottom sheets goes off screen and is removed from the DOM
$(selector).data("mBottomSheet").bind()
When the content gets dynamically updated the bind() function will search for anchor tags to hide or close the element on click
