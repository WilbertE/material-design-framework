Page structure

A navigation drawer is a menu which can slide in or is visible all the time on larger screens. It requires a special setup to make this work properly 

<!-- App bar or page headers should go first -->
               
<!-- Then comes the wrapper and the class to make the drawer stick on specific screen widths -->                 
<div class="navigation-drawer-wrapper navigation-drawer-wrapper--stick-laptop-up">
    
    <!-- Then the actual drawer-->
    <div class="navigation-drawer">
        <!-- Drawer content-->
    </div>
    <!-- Then the content of the page-->
    <content class="m-body">
        <!-- Page content -->
    </content>
</div>

Navigation drawer

The navigation drawer can have a head, dividers, labels and lists with buttons. 

<div class="navigation-drawer">
    <div class="navigation-drawer__header">
        <div class="navigation-drawer__header-title">Title</div>
        <div class="navigation-drawer__header-sub-title">Subtitle</div>
    </div>
    <div class="navigation-drawer__hr"></div>
    <div class="list navigation-drawer__list">
        <div class="list__item list__item--button">
            <button class="flat-button list__button" data-ripple="#cf5260">
                <i class="fas  fa-envelope list__icon"></i>Mailbox
            </button>
        </div>
    </div>
    <div class="navigation-drawer__label">Label</div>
</div>

Stick or slide

By default the navigation drawer is hidden off screen waiting for an event to slide it in. But it can be made stickable on specific (or all) screens using the classes navigation-drawer-wrapper--stick-laptop-up on the navigation-drawer-wrapper where 'laptop-up' can be replaced to match the requirements.

Initializing

To initialize the component use the following function, this will add a animation on clicks.

$(".navigation-drawer").mNavigationalDrawer();
Options

The following options are available:

onOpen: function(){}
Callback when the drawer opens
onClose: function(){}
Callback when the drawer closes
hamburger: null
A hamburger button selector to attach
Methods

Interacting with the component from outside:

$(".navigation-drawer").data("mNavigationalDrawer").open();
Opens the navigation drawer
$(".navigation-drawer").data("mNavigationalDrawer").close();
Close the navigation drawer

Example

Hide the drawer by resizing to or below tablet size and click this button 

$(".navigation-drawer").mNavigationalDrawer();
$(".js-show-navigation-drawer").on("click", function () {
    $(".navigation-drawer").data("mNavigationalDrawer").open();
});
