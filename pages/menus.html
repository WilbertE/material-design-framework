Menus

Menus open when the user clicks on a button and provide extra options. Menus are aware of the screensize and will open in the best possible way. They always live inside a button.

To create a link from an option add the attribute data-href with a destination and it will be automatically followed when an user clicks it. 

<button class="raised-button">
    Open menu
    <nav class="menu">
        <ul class="menu__options">
            <li class="menu__option" data-ripple="#cf5260"><i class="fal fa-edit"></i> Edit</li>
            <li class="menu__option" data-ripple="#cf5260"><i class="fal fa-save"></i> Save</li>
            <li class="menu__hr"></li>
            <li class="menu__option" data-ripple="#cf5260" data-href="menus.html"><i class="fal fa-sign-out-alt"></i> Log out</li>
        </ul>
    </nav>
</button>

Initializing

To initialize the component use the following function, this will add a animation on clicks.

$(selector).mMenu();
Options

The following options are available:

onSelect: function(item){}
The callback for when an option is clicked. The item parameter is the selected item.
Methods

Interacting with the component from outside:

$(selector).data("mMenus").open();
Opens the menu
$(selector).data("mMenus").close();
Closes the menu if visible
