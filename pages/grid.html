Grid

The grid is based on 12 columns with a default margin and padding of 8px. There are 4 breakpoints: Mobile (425px), Tablet (768px), Laptop (1024px) and pc (1440px).

Gutters

The default grid gutter is 8 but this can be changed to 16, 24 or 40 with the classes .grid--gutter-16, .grid--gutter-24 and .grid--gutter-40. The gutters only apply on the horizontal.

Sizing the cells

A cell is defined by using the class .grid__cell inside a .grid parent.
Sizes are defined by breakpoint names followed by a number for the column width between 1 and 12. Because the grid uses a mobile first method, all sizes without a name are used for mobile and up. For example .grid__cell--6 creates a cell with a width of 50% for mobile and up (all sizes).
The class .grid__cell--tablet-3 creates a cell with a width of 25% for tablet and bigger screen sizes. Combining these classes will create a responsive grid that changes depending on the screen size. If no size is defined then a the cell will be 100% width. 

<div class="grid">
    <div class="grid__cell grid__cell--tablet-6 grid__cell--laptop-3">
        Column
    </div>
</div>

Changing order

It is possible to change the order of the cells depending on the screen size. To do this the cells need to be wrapped inside a .grid__row after you can change the order with .grid__row--mobile-column-reverse and .grid__row--mobile-row-reverse. The classes can be used for all breakpoints. The order changes only for the specified screen size.

Offsets

Offsets can be created by using the class .grid__cell--offset-4 where the number sets the offset on the 12 column based grid. Like with the cell sizes the offset can also made depending on the breakpoint. So .grid__cell--tablet-offset-4 will offset the cell by 4 columns on tablet screens and larger.
