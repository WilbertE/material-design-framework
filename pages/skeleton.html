Skeleton lines

Skeletons are wireframes that organise the page when the content is still loading. When the content is loaded, the skeleton will be replaced by it. To create a div with skeleton lines use the class .skeleton-lines .skeleton-lines--1 where the number is the amount of lines.
A title should use .skeleton-title.

To hide something in skeleton view but keeping its space use skeleton-hidden, to hide it and have it no space use skeleton-clear.

Skeleton images

Image skeletons are images with a filled background. The class .skeleton-image will make that happen.

Skeleton components

Every component has a skeleton class.

The fieldset for an input or textarea should have the .skeleton-input class.
Helper text

A button should have the .skeleton-button

The fieldset for a dropdown should have the .skeleton-dropdown class.
