Autocomplete

An autocomplete will give the user suggestions about what it is trying to write. (examples start with Z) 

<fieldset>
    <input class="autocomplete" autocomplete="off" type="text" value="" id="input-label" />
    <label for="input-label">Label</label>
</fieldset>

Initializing

To initialize the component use the following function, this will add a animation on clicks.

$(".autocomplete").mAutocomplete(options);
Options

The following options are available:

array: []
A list of (static) items that fill the autocomplete (leave empty when using ajax requests in search option).
search: function (element, onSearchComplete){};
A method to create ajax calls to get suggestions. element holds the input element where element.val() will get the input value. When finished loading the reults the function should call onSearchComplete(element,array). Where 'array' is the array of search results.
minInputLength: 1
The minimum amount of characters to trigger a suggestion
maxItems: 3
The maximum amount of suggestions in the autocomplete list
searchDelay: 0
The delay a search gets started. If a user is typing fast it won't trigger a search for every character but waits for the user to stop typing.
onSelect: function(){}
Function that gets called when a user selects an item (clicks)
onFocus: function(element){}
Function that gets called when the input gets focus.
onBlur: function(element, itemExists){}
Function that gets called when the input loses focus. ItemExist will be true if the input matches a suggest item.
