Buttons

The most common used button types are Floating action buttons, raised buttons and flat buttons. 

<button class="raised-button">Button</button>
<button class="raised-button raised-button--disabled">Button</button>
<button class="flat-button">Button</button>
<button class="flat-button flat-button--disabled">Button</button>

Narrow buttons

Adding the class.flat-button--narrow or .raised-button--narrow creates a narrow button which gives buttons a very small side padding making it possible to push them more to the sides.

Initializing

To initialize the component use the following function, this will add a animation on clicks.

$.mButtons();
