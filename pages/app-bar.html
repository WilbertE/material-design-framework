App bar

The app bar is attached to the top of the screen (except in this example). It contains the main app functions and menus.

Page name
Contact
  • Edit
  • Save
  • Log out
<div class="app-bar">
    <!-- hamburger menu -->
    <div class="hamburger" data-ripple="#cf5260">
        <span class="hamburger-bar"></span>
    </div>
    <!-- Title -->
    <div class="app-bar__title">Page name</div>
    <!-- Right side -->
    <div class="app-bar__right">
        <!-- Icon button -->
        <div class="button-icon" data-ripple="#cf5260">
            <i class="fal fa-clock"></i>
        </div>
        <!-- Flat button -->
        <div class="flat-button" data-ripple="#cf5260">Contact</div>
        <!-- Icon button with menu -->
        <div class="button-icon" data-ripple="#cf5260">
            <i class="fal fa-ellipsis-v"></i>
            <nav class="menu">
                <ul class="menu__options">
                    <li class="menu__option" data-ripple="#cf5260"><i class="fal fa-edit"></i> Edit</li>
                    <li class="menu__option" data-ripple="#cf5260"><i class="fal fa-save"></i> Save</li>
                    <li class="menu__hr"></li>
                    <li class="menu__option" data-ripple="#cf5260" data-href="menus.html"><i class="fal fa-sign-out-alt"></i> Log out</li>
                </ul>
            </nav>
        </div>
    </div>
</div>

Hamburger menu

It is possible to insert a hamburger menu to the bar. It behaves the same as an icon button but should be attached to a sidemenu which it isn't by default. There are some configuration options.

  • - Place the hamburger menu inside a div with the class app-bar__right to place the hambuger menu on the right side
  • - Add responsive hidden classes (for example the class hidden-tablet-up) to hide the button on some screensizes
<div class="app-bar">
    <!-- hamburger menu -->
    <div class="hamburger" data-ripple="#cf5260">
        <span class="hamburger-bar"></span>
    </div>
    <!-- hamburger menu on the right side -->
    <div class="app-bar__right">
        <div class="hamburger" data-ripple="#cf5260">
            <span class="hamburger-bar"></span>
        </div>
    </div>
</div>

Buttons

Buttons behave like normal buttons that live inside the app bar. But with the class app-bar__full-button the button will be the same height as the app bar.

Page name
Dashboard
Mailbox
Contact
<div class="app-bar">

    <!-- Right side -->
    <div class="app-bar__right">

        <!-- Flat button -->
        <div class="flat-button app-bar__full-button" data-ripple="#fff">Dashboard</div>

    </div>
</div>

Hiding on scroll down

Using the class app-bar--hide-scroll will make the app bar dissapear when the user scrolls down and reappear when it scrolls down.

Initializing

To initialize the component use the following function.

$.mAppBar();

The hamburger menu button needs to be initialized on itself.

$.mHamburger();
